This App contains two mini React projects:

1. Background Color Changer

   * Clicking a color button changes the background color according to the selected button color.

2. Password Generator

   * Generates random secure passwords/tokens.

To run a specific project, go to the `main.jsx` file and comment/uncomment the required imports and function calls.
Comment the components you do not want to run and uncomment the component you want to display.
